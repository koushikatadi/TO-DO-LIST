# Discipline Tracker

Discipline Tracker is a habit and streak management web application built using Next.js and Firebase.  
The application helps users build consistent habits by tracking daily progress and maintaining streaks with secure, cloud-based storage.

---

## Features

- Google authentication using Firebase
- Secure user-based data access
- Habit creation, update, and deletion
- Automatic streak tracking
- Multi-device synchronization (mobile and desktop)
- Responsive and user-friendly interface

---

## Tech Stack

- Frontend: Next.js, React, TypeScript
- Styling: Tailwind CSS, ShadCN UI
- Backend: Firebase Authentication, Cloud Firestore
- Deployment: Vercel
